# お辞儀判子シミュレーター — Ojigi-in Drafting

「お辞儀ハンコ」——上司の印に向かって自分の印をすこし傾けて捺す日本の判子作法。
けれど「どれくらい傾ければいいのか」が分からない。

このサイトは、役職を **歯車** に見立てて礼の角度を一意に割り出す。

- **歯数 ＝ 序列**：偉い人ほど歯の多い大きな歯車（重く、回らない）
- **回転 ＝ お辞儀**：歯の少ない小さな歯車ほど大きく回り、深く礼をする
- **印は実体・歯車は抽象**：ハンコのサイズは全員同じ。歯車だけが序列で大小する

```
θ = θmax × (1 − 自分の歯数 / 最上位の歯数)
```

最上位は 0°（直立）、序列が下がるほど深く傾く。垂線からの傾き角として製図し、
計算結果のまま朱印が並んだ「お辞儀稟議書」を PNG で書き出せる。

## 使い方

`index.html` を開くだけ。単一HTMLファイルで完結（バックエンド不要）。
